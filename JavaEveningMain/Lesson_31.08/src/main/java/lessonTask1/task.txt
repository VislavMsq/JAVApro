Бизнес-логика: Реализовать приложение, в котором два потока будут обмениваться сообщениями.
Первый поток будет отправлять строку "Привет, второй поток!" второму потоку, а второй поток будет отправлять строку "Привет, первый поток!" первому потоку. После обмена сообщениями оба потока выводят принятое сообщение на экран.

Техническое задание:

Создать класс MessageExchange, который будет содержать два объекта Exchanger<String> для обмена строками между потоками.
Создать класс FirstThread, который будет запущен первым. Он создает строку "Привет, второй поток!" и обменивается этой строкой с вторым потоком.
Создать класс SecondThread, который будет запущен вторым. Он создает строку "Привет, первый поток!" и обменивается этой строкой с первым потоком.
Оба потока после обмена выводят на экран принятое сообщение.

public class MessageExchange {
    public static void main(String[] args) {
        Exchanger<String> exchanger = new Exchanger<>();

        Thread firstThread = new Thread(new FirstThread(exchanger));
        Thread secondThread = new Thread(new SecondThread(exchanger));

        firstThread.start();
        secondThread.start();
    }
}
