Задача: Управление заказами в интернет-магазине

Вы разрабатываете систему управления заказами в интернет-магазине. Создайте классы Order и Customer, а также опишите методы для взаимодействия с этими классами.
---------------------------------
Класс Order должен иметь поля:

orderId (String) - идентификатор заказа
totalAmount (double) - общая сумма заказа
items (List<OrderItem>) - список товаров в заказе
Методы класса Order:

++addItem(OrderItem item) - добавление товара в заказ
++removeItem(String itemName) - удаление товара из заказа по названию
++getItems() - получение списка товаров в заказе
calculateTotalAmount() - пересчет общей суммы заказа на основе товаров
--------------------------------
Класс OrderItem должен иметь поля:

itemName (String) - название товара
itemPrice (double) - цена товара
Методы класса OrderItem:

getItemName() - получение названия товара
getItemPrice() - получение цены товара
---------------------------------
Класс Customer должен иметь поля:

firstName (String) - имя клиента
lastName (String) - фамилия клиента
orders (List<Order>) - список заказов клиента
Методы класса Customer:

placeOrder(Order order) - размещение нового заказа клиентом
cancelOrder(String orderId) - отмена заказа по его идентификатору
getOrders() - получение списка заказов клиента
getTotalSpent() - подсчет общей суммы, потраченной клиентом на все заказы
