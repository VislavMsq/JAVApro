Задача 1:
Учет студентов и их оценок
Создайте класс Студент с полями:
    Имя,
    Фамилия,
    Возраст,
    Курс.
    enum status
Создайте класс предмет с полями:
    Предмет,
    Оценка.
Создайте класс ГенераторДанныхСтудентов, который будет возвращать случайные данные студентов и их оценок.
Создайте класс Университет, где будет храниться множество студентов и их оценки.

Реализуйте методы в Университете:
    Вывод всех студентов определенного курса.
    Средний балл студентов по определенному предмету.
    Поиск студентов, у которых средний балл выше заданной оценки.
    Группировка студентов по курсам.
    Сортировка студентов по возрасту.
